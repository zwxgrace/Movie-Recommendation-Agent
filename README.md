# 🎬 LLM驱动的电影推荐系统 (LLM-Powered-Semantic-Movie-Recommendation-System)

## 📖 项目背景

在当今的流媒体时代，电影推荐系统在提升用户体验方面扮演着至关重要的角色。本项目旨在解决海量内容下用户难以发现心仪影视作品的痛点。

* **海量内容库 (Massive Content Libraries)**
  现代流媒体平台（如 Netflix、Amazon Prime Video 和 Disney+）拥有成千上万的电影和电视节目。虽然庞大的资源库提供了丰富的选择，但也使得用户的内容发现（Content Discovery）变得越来越困难。
  
* **推荐系统的重要性 (Importance of Recommendation Systems)**
  数据表明，Netflix 上约 **80%** 的观看内容都是由推荐系统驱动的。个性化推荐能够显著提升用户的平台参与度（Engagement）和留存率（Retention）。

## 🚀 核心推荐算法 (Common Recommendation Approaches)

本项目主要探索并实现了以下几种主流的推荐机制：

1. **协同过滤 (Collaborative Filtering)**
   基于相似用户的历史行为（如观看记录、评分等）来为目标用户生成推荐。
   
2. **基于内容的过滤 (Content-Based Filtering)**
   基于电影的元数据（如电影类型、导演、演员阵容等）进行特征匹配，向用户推荐与他们过去喜欢的电影相似的内容。
